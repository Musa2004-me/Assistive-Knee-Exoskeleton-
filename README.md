The objective of this project is to develop a complete OpenSim-based simulation framework for a 
lower-limb knee exoskeleton. The project integrates a musculoskeletal gait model, a CAD-based 
exoskeleton, and a MATLAB PID controller. The exoskeleton actuator applies assistive torque to 
the right knee joint, and the effects of assistance are evaluated through muscle activation analysis, 
muscle force analysis, and metabolic energy estimation. 


The primary software tools used in the project are: 


• OpenSim for musculoskeletal modeling and biomechanical simulation. 


• MATLAB for controller implementation and closed-loop simulation. 


• CAD software for exoskeleton geometry editing and alignment.
